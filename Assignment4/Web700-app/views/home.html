Welcome to the WEB700 app!

This website has been made to demonstrate HTML5 principles and techniques, running routes in a Express JS backend.

Website Features

This website consists of 3 static views and 5 routes to supply data using JSON format.

Static routes

  • / - Home page
  • /about - About the author
  • /htmlDemo - Demonstration of HTML5 features

Data routes

  • /students - Return all Students.
  • /students?course={id} - Return students enrolled in the course with id {id}.
  • /students/{student_id} - Return the student with id {student_id}.
  • /tas - Return all TAs.
  • /courses - Return all available courses.

About HTML5

This website leverages HTML5, which is the most recent version of HTML (Hyper Text Markup Language), a widely used protocol to structure and present content on the World Wide Web. HTML5 stands out from previous versions by providing support for modern media formats, improved responsiveness and better integration with other technologies.
